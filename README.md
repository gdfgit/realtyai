# 🏠 Realty AI

AI-powered real estate assistant that searches homes across Zillow, Realtor.com, Redfin, Homes.com, LoopNet, Crexi & BizBuySell.

## Features
- Residential, Commercial & New Construction search
- Email verification registration
- Mortgage calculator & CMA analysis
- Voice search & file upload
- Neighborhood info (restaurants, schools, hospitals, gyms, parks)

## Tech Stack
- React
- Supabase (database)
- EmailJS (email verification)
- Tavily (web search)
- Netlify (hosting)
